# ScheduleMate · 课表通

拖入任意学校课表 Excel，自动识别格式，一键筛选教师、智能排课、计算课时、导出结果。

## 功能

- **智能识别** — 自动识别五种课表格式，无需手动调整
- **教师筛选** — 输入姓名（支持模糊匹配），课程自动黄色高亮
- **个人课表** — 勾选「只显示本人课程」，只看自己的每周安排
- **课时计算** — 按月统计折合课时，区分正课/自习/晚自习，支持放假和调休
- **智能排课** — 导入班级、教师、课时信息，一键生成全校课表，支持科目限制、日上限、晚自习模式等规则
- **导出** — 导出带样式的 Excel，或打印为 PDF

## 使用方法

### 课表筛选

1. 浏览器打开 `index.html`
2. 拖入 Excel 课表文件
3. 输入教师姓名，点击「确定」
4. 使用「📊 课时计算」统计月度工作量

### 智能排课

1. 切换至「课表排课」标签
2. 下载模板，填写班级、科目、教师、课时信息
3. 上传配置，设定排课规则
4. 点击「开始排课」一键生成课表

## 环境要求

- Chrome / Edge / Firefox / Safari 现代浏览器
- 首次使用需联网（加载 SheetJS 库）
